Monefy uses SQLite3 for its database. When exporting this database, Monefy encrypts the SQLite3 database using a hardcoded key to make the export unusable unless it's imported directly into Monefy. This tool lets you decrypt a backup to get the original SQLite3 database, or encrypt a SQLite3 database back into a Monefy-compatible backup.
Both operations happen completely in the browser and no data is transmitted anywhere.
How to export from Monefy?
Open Monefy > Click the 3 dot menu on the top right > Settings > Create data backup
